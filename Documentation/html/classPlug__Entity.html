Detailed Description

Wrapper for acces entities from plugins.

Wrapper class for create, acces and modify entities from plugins. TODO: terminate acces function -> getData() terminate implementation of modify function -> updateData() terminate implementation of create function (ctor called from document) can't create entities:

  • atomic = CONSTRUCTIONLINE, OVERLAYBOX, SOLID,
  • container = INSERT, POLYLINE, SPLINE, HATCH, DIMLEADER, DIMALIGNED, DIMLINEAR, DIMRADIAL, DIMDIAMETRIC, DIMANGULAR, verify destructor if needed or not to delete entity
