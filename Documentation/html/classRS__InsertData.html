nameThe name of the block used as an identifier.

scaleFactorScale factor in x / y.

colsNumber of cols if we insert a whole array.
rowsNumber of rows if we insert a whole array.
spacingSpacing between rows and cols.
blockSourceSource for the block to insert if other than parent. Normally blocks are requested from the entity's parent but the block can also come from another resource. RS_Text uses that to share the blocks (letters) from a font.
updateModeRS2::Update will update the insert entity instantly RS2::NoUpdate will not update the insert. You can update it later manually using the update() method. This is often the case since you might want to adjust attributes after creating an insert entity.
