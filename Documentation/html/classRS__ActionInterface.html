Detailed Description

This is the interface that must be implemented for all action classes. Action classes handle actions such as drawing lines, moving entities or zooming in.

Inherited from QObject for Qt translation features.

Author
Andrew Mustun

void RS_ActionInterface::setStatus ( int  status)
virtual

Sets the current status (progress) of this action. The default implementation sets the class variable 'status' to the given value and finishes the action if 'status' is negative.

Parameters
statusStatus number. It's up to the action implementor what the action uses the status for. However, a negative status number finishes the action. Usually the status of an action increases for every step of progress and decreases when the user goes one step back (i.e. presses the right mouse button).
