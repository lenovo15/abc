\subsection{Detailed Description}
Abstract class (interface) for the creation of new entities. Inherit your class which takes care of the entities in the processed D\-X\-F file from this interface.

Double arrays passed to your implementation contain 3 double values for x, y, z coordinates unless stated differently.
