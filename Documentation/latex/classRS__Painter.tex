\subsection{Detailed Description}
This class is a common interface for a painter class. Such a class will in it's implementation be responsible to paint lines, arcs, ... in widgets. All angles in rad.

Note that this is just an interface used as a slot to communicate with the Libre\-C\-A\-D from a G\-U\-I level. This does not contain any Qt or platform specific code.
