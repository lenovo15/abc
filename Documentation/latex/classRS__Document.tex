\subsection{Detailed Description}
Base class for documents. Documents can be either graphics or blocks and are typically shown in graphic views. Documents hold an active pen for drawing in the Document, a file name and they know whether they have been modified or not.

Andrew Mustun 
\end{DoxyAuthor}
